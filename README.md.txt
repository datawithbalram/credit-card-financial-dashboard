Credit Card Financial Dashboard
Project Overview
This project presents a comprehensive Credit Card Financial Analysis Dashboard developed using Power BI. The dashboard provides insights into customer behavior, transaction patterns, revenue generation, and spending trends. The objective of the project is to transform raw financial data into meaningful business insights that support data-driven decision-making.
Tools & Technologies Used
•	Power BI
•	SQL
•	Power Query
•	DAX
•	Microsoft Excel
Key Features
•	Interactive dashboards with dynamic filters and slicers
•	Revenue, transaction amount, and interest analysis
•	Customer segmentation by occupation, education, age group, and marital status
•	Card category and transaction mode analysis
•	Week-over-Week revenue tracking using DAX measures
•	Business insights and performance monitoring
Key Metrics
•	Total Revenue: 56.52M
•	Total Transaction Amount: 45.53M
•	Total Interest Earned: 7.98M
•	Total Transaction Count: 667K
Business Insights
•	Blue Card customers generated the highest revenue contribution.
•	Q4 recorded the highest revenue and transaction volume.
•	Businessmen and White-collar professionals were the top revenue-generating customer segments.
•	High-income customers contributed significantly to overall revenue.
•	Bills, Entertainment, and Fuel were the leading expenditure categories.
•	Swipe transactions accounted for the majority of card usage.
•	Graduate customers generated the highest revenue among education groups.
